# Business Sales Performance Analytics
## Future Interns - Data Science & Analytics | Task 1

---

## 📌 About the Project
This project analyzes business sales data to uncover revenue trends, 
top-selling products, category performance, and regional profitability 
using Tableau Public.

## 🛠️ Tool Used
- Tableau Public

## 📁 Dataset Used
- Sample Superstore Sales Dataset (Kaggle)


## 📊 Live Dashboard
🔗 [https://public.tableau.com/shared/6SG22BH3S](https://public.tableau.com/app/profile/jiya.jiya2812/viz/BusinessSalesPerformance-Jiya/BusinessSalesPerformanceDashboard?publish=yes)

---

## 🔍 Key Insights
- Technology is the highest revenue category ($836K)
- Sales peak every November — strong seasonal demand pattern
- Canon imageCLASS 2200 is the #1 top-selling product
- West region is the most profitable ($108K profit)
- Central region has the lowest profit ($39K) — needs business attention
